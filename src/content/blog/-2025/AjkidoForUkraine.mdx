**{frontmatter.description}**
<br/>
Після початку повномасштабної війни для багатьох українських клубів стало значно складніше продовжувати стабільну практику, організовувати семінари та підтримувати розвиток айкідо.
<br/>
<br/>
Саме в цей період виник міжнародний проєкт <a href="https://www.aikidoforukraine.org/" target="_blank">Aikido for Ukraine</a>, який об’єднав друзів, викладачів та айкідо-клуби з різних країн навколо підтримки українських практиків.
<br/>
<br/>
Наша академія також стала частиною цього проєкту.
<br/>
<br/>
Завдяки підтримці наших друзів та партнерів з Європи ми отримали можливість продовжувати тренування, організовувати семінари, дитячі заходи та активно розвивати айкідо навіть у дуже непростий для країни час.
<br/>
<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779640743/afu7_b559fe.jpg" alt="" title=""/>

<br/>
Для нас це значно більше, ніж просто допомога.
<br/>
<br/>
Це відчуття того, що ми є частиною великої міжнародної спільноти, де люди підтримують одне одного не лише на татамі, а й у реальному житті.
<br/>
<br/>
Завдяки цьому проєкту наші учні отримують можливість подорожувати Європою, брати участь у цікавих міжнародних заходах та знайомитися з практикою айкідо в різних країнах.
<br/>
<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779640743/afu6_tt7d5o.jpg" alt="" title=""/>

<br/>
Водночас багато майстрів та викладачів приїжджають в Україну, щоб підтримати нас, провести тренування, поділитися досвідом та допомогти продовжувати розвиток навіть у складних умовах.
<br/>
<br/>
Ми також відчуваємо фінансову підтримку від цього проєкту, і дуже вдячні всім, хто долучається до цієї ініціативи.
<br/>
<br/>
Саме така допомога дає можливість не лише не зупинятися, а навпаки — активно рухатися вперед.
<br/>
<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779640743/afu2_hxci3l.jpg" alt="" title=""/>

<br/>
За останні роки ми дуже сильно прогресували у своєму розвитку айкідо, отримали багато нового досвіду, нових знайомств та можливостей для навчання.
<br/>
<br/>
І велика частина цього стала можливою саме завдяки людям, які продовжують підтримувати Україну та українську айкідо-спільноту.
<br/>
<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779640743/afu4_grvfie.jpg" alt="" title=""/>

<br/>
Ми щиро вдячні всім клубам, організаціям, викладачам та друзям, які залишаються поруч із нами.
<br/>
<br/>
Попереду ще багато спільних тренувань, семінарів та міжнародних зустрічей.
<br/>
<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779640743/afu3_odkfzp.jpg" alt="" title=""/>

<br/>
І ми раді бути частиною цього великого міжнародного проєкту.
<br/>
<br/>
