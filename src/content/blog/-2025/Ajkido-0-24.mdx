**{frontmatter.description}**
<br/>
14 березня 2015 року в Ужгороді відбувся перший в Україні міжнародний 24-годинний айкідо-марафон.
<br/>
<br/>
Це був безпрецедентний для українського айкідо формат, який об’єднав учасників та викладачів із різних країн навколо ідеї безперервної практики протягом цілої доби.
<br/>
<br/>
У межах марафону 12 викладачів із 4 країн провели 24 тренування поспіль у форматі non-stop.
<br/>
<br/>
Для багатьох учасників це стало не лише фізичним випробуванням, а й можливістю перевірити власну витривалість, дисципліну та силу волі.
<br/>
<br/>
<br/>
Ідея проведення такого заходу була перейнята від наших друзів зі Словаччини, які вже мали досвід організації подібних марафонів.
<br/>
<br/>
Саме такі події дозволяють по-іншому подивитися на практику айкідо — не лише як на набір технік, а як на процес роботи над собою, взаємодії та подолання власних меж.
<br/>
<br/>

> «Iстинна перемога — це перемога над самим собою»
> <br/>
> Моріхей Уесіба
