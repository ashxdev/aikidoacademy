**{frontmatter.description}**
<br/>
Є семінари, на які завжди приємно повертатися. Саме такими для нас уже давно стали тренування у Кошицях, які організовують наші друзі з <a href="https://www.instagram.com/martinfrankovic_aikidoakademia/" target="_blank">Aikido Akademia</a>.
<br/>
<br/>
Цього разу від нашого доджо на семінарі були присутні <a href="https://www.instagram.com/timeia.markina/" target="_blank">Тімея Маркіна</a> та <a href="https://www.instagram.com/myxomop44/" target="_blank">Іван Мула</a>, і поїздка виявилася надзвичайно цінною та насиченою.
<br/>
<br/>
Семінар <a href="https://aikido-brunogonzalez.com/en" target="_blank">Bruno Gonzalez</a> залишив після себе багато вражень, нових ідей та важливих деталей у роботі. Після повернення учасники привезли із собою цілий багаж нових знань і з великим натхненням поділилися ними вже на наших тренуваннях.
<br/>
<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779558809/bruno26_sy8dbe.jpg" alt="" title=""/>

<br/>
Саме такі поїздки дозволяють підтримувати живий зв’язок між доджо, розширювати власне бачення практики та продовжувати розвиток навіть через невеликі зустрічі й семінари.
<br/>
<br/>
Ми дуже вдячні проєкту <a href="https://www.aikidoforukraine.org/" target="_blank">Aikido for Ukraine</a> за можливість подорожувати, тренуватися та відвідувати настільки цінні семінари у цей непростий час.
