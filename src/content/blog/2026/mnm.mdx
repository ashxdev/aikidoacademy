**{frontmatter.description}**
<br/>
Можна рухатись у групі — і все одно йти не туди.
<br/>
Але коли люди свідомо обирають бути поруч, підтримувати та приїжджати до тебе — саме це визначає напрямок.
<br/>
<br/>
Цими вихідними ми провели черговий молодіжний семінар з айкідо.
Тренування проводили <a href="https://www.facebook.com/norbert.hochstrasser" target="_blank">Norbert Hochstrasser</a> та <a href="https://www.instagram.com/martinfrankovic_aikidoakademia/" target="_blank">Martin Frankovic</a>, працюючи з нашими учнями та ділячись власним досвідом.
<br/>
<br/>
Для дітей це значно більше, ніж просто практика.
Це справжній зв’язок і відчуття приналежності до чогось більшого.
<br/>
<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779558700/mnm263_lycmi2.jpg" alt="" title=""/>

<br/>
У такі часи подібна підтримка має особливе значення.
Вона нагадує нам, що ми не самі — навіть якщо нас розділяють кордони.
<br/>
<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1780473825/mmnmm8_eviebf.jpg" alt="" title=""/>
Ми рухаємось вперед разом.
І це відчувається правильним саме через те, що ми робимо на цьому шляху.
<br/>
<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1780473825/mnm268_muelag.jpg" alt="" title=""/>

<br/>

<img class="centered" src="https://res.cloudinary.com/du5rp9nvc/image/upload/v1779558696/mnm265_i8r4sf.jpg" alt="" title=""/>

<br/>
Дякуємо всім, хто приїхав.
<br/>
Особлива подяка нашим друзям з <a href="https://www.kyiv-aikido.club/contacts" target="_blank">Kyiv Aikido Club</a> та <a href="https://www.instagram.com/varya.smkh/" target="_blank">Varvara Samokhina</a> — бережіть себе та залишайтеся в безпеці.
<br/>
<br/>
Дякуємо <a href="https://www.aikidoforukraine.org/" target="_blank">Aikido for Ukraine</a> — за підтримку, яку ми відчуваємо у складні часи.
<br/>
<br/>
І дякуємо Збройним Силам України — завдяки вам ми можемо продовжувати тренування, збиратися разом та безпечно приймати друзів.
<br/>

Під час цього семінару наші учні також складали іспити на кю — від 5 до 1 кю. Усі чудово впоралися зі своїми випробуваннями.
<br/>
<br/>
Також ми вручили дипломи тим, хто раніше успішно склав іспит на чорний пояс першого дану.
Для всіх нас це був дуже важливий і гордий момент.
<br/>
<br/>
Крок за кроком — саме так ми ростемо разом.
<br/>
